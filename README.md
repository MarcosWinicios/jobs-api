# jobs-api
## Está é uma API criada com o propósito de fornecer informações sobre vagas de emprego para desenvolvedores. Este é um projeto baseado na plataforma <a href="https://programathor.com.br/">Programathor</a>

### Tecnologias
- Java(JDK 11)
- Maven
- Spring Boot
- Spring Data JPA
- Postgres
